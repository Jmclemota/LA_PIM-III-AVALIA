# Diagramas de Casos de Uso

---

Este diretório contém os **diagramas de casos de uso** do projeto *Avalia+*, desenvolvidos como parte do PIM III.

Os diagramas de casos de uso representam as **funcionalidades do sistema** a partir da perspectiva dos usuários, mostrando como cada tipo de usuário interage com o sistema.

---

## Objetivo

Demonstrar as funcionalidades disponíveis no sistema, evidenciando:

- As ações que cada usuário pode realizar  
- A interação entre usuários e o sistema  

  ---
  
## Ferramenta Utilizada

Os diagramas foram desenvolvidos utilizando a ferramenta **Astah**, permitindo a modelagem UML de forma clara e estruturada.
