# Diagramas de Classes

---

Este diretório contém os **diagramas de classes** do projeto *Avalia+*, desenvolvidos como parte do PIM III.

Os diagramas de classes representam a **estrutura do sistema**, incluindo suas classes, atributos, métodos e os relacionamentos entre elas.

---

## Objetivo

Demonstrar de forma clara a organização interna do sistema, evidenciando:

- As principais classes do sistema  
- Seus atributos e métodos  
- Os relacionamentos entre as classes (associação, herança, dependência, etc.)  

---

## Estrutura

Os diagramas foram organizados de acordo com os principais tipos de usuários do sistema:

-  Aluno  
-  Professor  
-  Administrador  

---

##  Ferramenta Utilizada

Os diagramas foram desenvolvidos utilizando a ferramenta **Astah**, que permite a modelagem UML de forma visual e organizada.
