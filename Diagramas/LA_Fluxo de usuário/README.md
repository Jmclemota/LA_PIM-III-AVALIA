# Fluxo de Usuário – Projeto Avalia+

Esta pasta contém o Fluxo de Usuário (User Flow) do projeto Avalia+.

O fluxo de usuário representa o caminho percorrido pelos usuários dentro
do sistema, desde o acesso inicial até a utilização das funcionalidades disponíveis.

## Objetivo do Fluxo de Usuário

O fluxo de usuário tem como objetivo mapear a navegação no sistema,
facilitando a organização da interface, melhorando a experiência do usuário e auxiliando no desenvolvimento das funcionalidades.
