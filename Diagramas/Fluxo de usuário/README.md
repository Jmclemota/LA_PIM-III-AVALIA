# Fluxo de Usuário

---

Este diretório contém os **diagramas de fluxo de usuário** do projeto *Avalia+*, desenvolvidos como parte do PIM III.

Os fluxos de usuário representam o **caminho percorrido pelos usuários dentro do sistema**, desde o acesso inicial até a execução de suas ações.

---

## Objetivo

Mapear a navegação do sistema, facilitando a compreensão de:

- Como o usuário interage com o sistema  
- A sequência de ações realizadas  
- A experiência do usuário (UX)  

---

## Ferramenta Utilizada

Os diagramas foram desenvolvidos utilizando a ferramenta **Astah**, permitindo a modelagem visual dos fluxos de navegação.
